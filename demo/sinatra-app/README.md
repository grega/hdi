# sinatra-app

A lightweight API built with Sinatra.

## Prerequisites

```bash
brew install ruby
```

## Installation

```bash
bundle install
cp .env.example .env
```

## Running

```bash
bundle exec ruby app.rb
```

## Testing

```bash
bundle exec rake test
```

## API Response

```json
{
  "status": "ok",
  "data": []
}
```
